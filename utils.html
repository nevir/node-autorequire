Helper to allow autorequire calls to support the same kind of relative pathing that require performs.

There has to be a better way of doing this than parsing a stack trace, though.

The offset indicates how many calls we should go back in the stack to find a caller. 0 is the function that is calling getCallingDirectoryFromStack.
